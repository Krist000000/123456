## 验证真伪智能体

### 功能简介

验证真伪智能体是一个基于网络搜索的信息真实性验证工具，旨在帮助用户验证所收到信息的真伪。

### 核心特性

- ✅ **联网搜索验证**：必须联网检索，搜集权威来源资料
- 🔬 **交叉比对分析**：对信息细节进行仔细核查，不放过任何细节
- 📊 **权威来源依据**：仅基于公开、权威的网络信息进行核查
- ✨ **明确真伪结论**：清晰说明信息真伪，附带关键核查来源

### 工作流程

1. **接收信息** - 接收用户发送的待验证信息
2. **联网检索** - 启动联网搜索，搜集权威来源的相关资料
3. **交叉比对** - 对比多个来源，分析信息的一致性和可靠性
4. **输出结论** - 给出明确的真伪判断及详细的核查依据

### 使用限制

- 仅能基于**公开、权威的网络信息**进行核查
- 无法获取**未公开的私密数据**或内部信息
- 验证结果依赖于公开可查信息源的可靠性

### 使用方法

1. 打开 `verification-agent.html` 文件
2. 在输入框中输入需要验证的信息
3. 点击"开始验证"按钮
4. 等待系统联网搜索和分析
5. 查看验证结果和权威来源

- 清晰的用户界面和工作流程可视化
