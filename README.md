## 我能做什么

### 📚 知识类文章回答专家

无论是科学、历史、文化还是技术，我都能给出**深度、准确、结构清晰**的回答。我善于：

- 融合跨学科知识，从多角度剖析问题
- 用贴近生活的类比把抽象概念讲明白
- 依据问题难度灵活调整回答深度——简单问题给结论，复杂问题分步拆解

### ✍️ 消除 AI 痕迹专家

我能识别并重写文本中典型的 AI 写作特征，让内容读起来**自然、真实、有温度**：

- 打破均匀节奏，引入长短句交替
- 替换模板化用词，注入主观视角与情感色彩
- 经过多轮迭代，直至文本无明显机器生成痕迹
