# 信息验证智能体 (Information Verification Agent)

## 📋 简介

信息验证智能体是一个基于人工智能的信息真伪验证系统，能够：

- ✅ **联网搜索验证**：自动搜索权威来源，核查信息真伪
- 🔍 **交叉比对分析**：对比多个来源，确保信息准确性
- 📊 **详细验证报告**：提供完整的验证依据和来源链接
- 🎯 **可信度评级**：标注信息可信度和验证置信度

## 🚀 核心功能

根据问题陈述要求，本系统实现了以下核心功能：

1. **核心作用**：帮助验证所发信息的真伪，必须联网搜索验证
2. **联网查询能力**：对提供的信息进行仔细核查，不放过任何细节
3. **工作流程**：
   - 接收待验证信息
   - 启动联网检索，搜集权威来源资料
   - 交叉比对信息细节
   - 输出明确的真伪结论及核查依据
4. **限制说明**：仅基于公开、权威的网络信息进行核查
5. **输出形式**：清晰说明信息真伪，附带关键核查来源或比对要点

## 🎯 验证标准

### 结论类型

- **真实** - 信息得到多个权威来源证实，置信度高
- **部分真实** - 信息基本属实但有细节偏差
- **存疑** - 来源信息不一致或不充分
- **可能虚假** - 多数来源与信息相悖
- **无法充分验证** - 缺乏足够的公开信息

### 置信度评级

- **高** - 有充分的权威来源支持结论
- **中** - 有一定来源支持但需要更多验证
- **低** - 来源不足或质量不高

## ⚠️ 注意事项

1. **网络搜索集成**：当前版本使用模拟搜索结果。实际部署时需要集成真实的搜索API：
   - Google Search API
   - Bing Search API
   - DuckDuckGo API
   - 专业事实核查API（FactCheck.org, Snopes等）

2. **数据来源**：系统仅能基于公开、权威的网络信息进行核查，无法获取未公开的私密数据

3. **准确性**：验证结果的准确性取决于可获取的公开信息质量和数量

4. **更新需求**：建议定期更新权威来源列表，确保验证准确性
